## 発生したエラー

[Ghostty](https://ghostty.org/)を使用して[Claude Code](https://code.claude.com/docs/ja/overview)をCtrl Cで終了するとその後Ctrlが効かなくなる事象に遭遇した。

`cat`コマンドを使って確認するとCtrl pを入力した際に以下のようになっている。

```shell
cat
[[112;5u
```

### 環境

- Ubuntu 24.04.4 LTS
- ghostty: 1.2.3
- zsh 5.9
  - vi-mode
- Claude Code: 2.1.84

---

## 解決方法

`printf '\e[>0u'`すると直ることがわかった。

Claude Code終了時にhooksをすることができなかったので、`~.zshrc`に以下を設定した。

```~/.zshrc
# Claude Code終了後にKitty keyboard protocolをリセット
claude() {
  command claude "$@"
  printf '\e[>0u'
}
```
