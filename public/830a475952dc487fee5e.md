## これは何?

Ubuntu 24.04 LTSを使い始めたところ、System Extensionsが増えたなと感じたので、それで何ができるのか簡単にまとめた。

System Extensionsをオンオフしてもパッと見何ができるかわかりにくいものもあったので。

---

## 環境

```shell
cat /etc/os-release                                       1
PRETTY_NAME="Ubuntu 24.04.3 LTS"
NAME="Ubuntu"
VERSION_ID="24.04"
VERSION="24.04.3 LTS (Noble Numbat)"
VERSION_CODENAME=noble
ID=ubuntu
ID_LIKE=debian
HOME_URL="https://www.ubuntu.com/"
SUPPORT_URL="https://help.ubuntu.com/"
BUG_REPORT_URL="https://bugs.launchpad.net/ubuntu/"
PRIVACY_POLICY_URL="https://www.ubuntu.com/legal/terms-and-policies/privacy-policy"
UBUNTU_CODENAME=noble
LOGO=ubuntu-logo
```

## System Extensionsとできること

### Apps Menu

左上にAppsという項目が追加されて、カテゴリ別でアプリが表示されるようになるもの。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/057638a6-7936-456c-8b7f-445c4090fb31.png)

### Auto Move Windows

アプリをどのモニターで開くかデフォルト設定を決められる。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/7ea93b37-784f-4619-87e1-b9eac1f0f71c.png)

### Desktop Icons NG (DING)

名前的にDesktopにアイコンを作れなくする拡張なのかと思っていたが、逆でオンにするとデスクトップにアイコンを作れるようになる

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/9637e95c-b2c7-4ab8-9d8a-fb350cb979f2.png)

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/d226b699-5ccb-4cc8-8c71-d2a5c3582c1c.png)

### Launch new instance

アプリをクリックして起動する際に常に最新のインスタンスを作る

### Light Style

明るいテーマをデフォルトにする。

### Native Window Placement

superキーを押したときの一覧表示が最適化される。
アプリをたくさん開くかつ、画面が少ない人にはおすすめ。
