## うまく行った方法

### localeの変更

https://dflabo-jp.tech/japanese-localization-of-kali-linux/

この通り実施。ただし、この手順をやっただけではsuper spaceで日本語入力に切り替えできなかったので、後述のmozcインストール手順を実施した。

## mozcインストール

apt installがうまくいかなかったので一時的にdebianパッケージを使用してインストールした。

```shell
echo "deb http://deb.debian.org/debian bookworm main" | sudo tee /etc/apt/sources.list.d/debian.list
sudo apt update
sudo apt install ibus-mozc
reboot
```

再起動するとOSの言語が日本語になっている。
