## Ubuntu 22.04 LTSから Ubuntu 24.04系へアップデートした理由

https://ghostty.org/docs/install/binary#snap

上記ページを見てinstallを試みたのですが、`.deb`ファイル経由だと22系はサポート対象外だったため、これを機に上げました。

一応22系のままでも`sudo snap install ghostty --classic`すればインストールできましたが。

Ubuntu 24系へのアップデートはコマンド一つでできた。

```shell
do-release-upgrades
```

## Ghostty install (Ubuntu 24.04 LTS)

[公式のインストール手順](https://ghostty.org/docs/install/binary#snap)だと権限不足だったので`sudo`だけつけている。

```shell
sudo /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/mkasberg/ghostty-ubuntu/HEAD/install.sh)"\n
```

## Solarizid Dark化

https://ghostty.org/docs/features/theme

以下のコマンドで使用できるcolor schemeが選べるので、選択して`~/.config/ghostty/config`に記載するだけ。

```shell
ghostty +list-themes
```
Solarized Darkは明るめのものと暗めのものの2種類ある。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/554a5201-ee1c-4b55-802c-e902d10d9595.png)

```~/.config/ghostty/config
theme = Solarized Dark Higher Contrast
background-opacity = 0.85 # ついでに背景を透明にしておいた。
```
