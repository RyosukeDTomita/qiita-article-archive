## これは何?

あけましておめでとうございます。今年もよろしくお願いいたします。

2025年10月からHaskellで競技プログラミングを始め、論文を読んで管理しておきたいというモチベーションが湧いてきました。

この記事では、論文管理のために使う、[Zotero](https://www.zotero.org/)を[Obsidian](https://obsidian.md/)に連携する方法についてまとめます。

---

## 環境

- Ubuntu 22.04
    - Obsidian: 1.10.6
    - Zotero: 7.0.24

### 前提条件

- Obsidianのインストールがされていること

---

## Zoteroインストール

Ubuntu公式ストアからGUI経由でインストールした。

### zotero-better-bibtexインストール

- [公式のGitHubのRelease](https://github.com/retorquere/zotero-better-bibtex/releases/tag/v7.0.70)から`.xpi`ファイルをダウンロードする。

    ```shell
    wget https://github.com/retorquere/zotero-better-bibtex/releases/download/v7.0.70/zotero-better-bibtex-7.0.70.xpi
    ```
    :::note warn
    自分の環境ではReleaseページからクリックしてダウンロードするとうまく行かなかったので`wget`からダウンロードした。

- Zoteroに拡張機能をインストールする

    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/d187814e-a34e-4fc5-95f3-7c3a18f5f07a.png)
    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/9e08972c-42a6-42bb-98d3-df05659513bd.png)

---

## ObsidianにCitationsをインストールしてセットアップする

- [Citations](https://github.com/hans/obsidian-citation-plugin)はObsidian経由でInstallできる。

    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/7a62b348-e27e-408d-8a69-f6417747e747.png)

- Zoteroから`.bib`をExportする
    - `Export Library...`
        ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/d5e6f1d2-cc03-45de-a13a-445c5c64737d.png)
    - `Better BiBTex`を選択(残りは適当)
        ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/a77c6ae4-0f7d-4622-a5c8-adb3adcd9e40.png)
    - Export場所を選択してExportする

- Citiationsの設定
    - `Citation database format`を`BibLaTeX`
    - `Citiation database path`にZoteroからExportした`.bib`のパスを設定 例: `Literature/zotero/My Library/My Library.bib`
    - テンプレートを編集したければ編集する

    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/ea1fcf58-7af6-4167-86de-0de42eaa4207.png)

## 動作確認

### 論文のメタデータをObsidian側に自動作成

1. 新しいノートを作成する
2. Ctrl Pでコマンドパレットを開き、{Citations: Insert Literature note content in the current pane`を選択する

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/4373bbda-2ed5-4fab-802f-e93add88d708.png)

Propertiesに論文データが挿入された!!

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/bcc798cd-0fd6-4fe7-89e5-15dfc46e310a.png)


### 英語翻訳

1. [Translate for Zotero](https://github.com/windingwind/zotero-pdf-translate)をダウンロード 
    ```shell
    wget https://github.com/windingwind/zotero-pdf-translate/releases/download/v2.3.16/translate-for-zotero.xpi
    ```
2. 設定から英語→日本語に設定する: `Translate From English (United States) To Japanese(Japan)`
    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/74ef2b0b-bcb2-424c-bf9b-388f94328a20.png)
    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/ead10cd1-b797-46cb-9d4b-c425c7e37ffe.png)
3. 論文を開いてハイライトすると翻訳される
    ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/7fdc6ad5-5c25-49b2-9b4b-b057784ef21b.png)

:::note info
無料でGoogle APIを叩いてGoogle Translateが使えるようだが、ChatGPTのAPIも指定可能。
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3718390/9585cded-d2f8-4b93-965f-891834b9de8b.png)

:::
